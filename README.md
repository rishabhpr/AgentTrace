# AgentTrace

Structured tracing format for AI agent execution flows.

## Specification

- [`agenttrace-spec.html`](agenttrace-spec.html) - Interactive HTML specification
- [`agenttrace-spec.pdf`](agenttrace-spec.pdf) - PDF version

## Overview

AgentTrace defines a standard format for capturing and analyzing agent execution traces, enabling:

- **Debugging** - Step-by-step visibility into agent decision-making
- **Visualization** - Render agent workflows as diagrams
- **Analysis** - Query and aggregate trace data for insights
- **Replay** - Reconstruct and replay agent sessions
